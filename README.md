# TikTok Favorites Downloader

Цей додаток дозволяє завантажувати всі відео з розділу "Збережене" (Favorites) вашого акаунту TikTok за нікнеймом.

## Оновлення: Новий метод авторизації
Ми перейшли на використання **Playwright** для автоматизації браузера. Тепер вам **не потрібно** вручну шукати `msToken` або інші кукі. Достатньо один раз увійти в акаунт у вікні браузера, що відкриється.

## Особливості
- **Автоматична авторизація:** Сесія зберігається в папці `browser_profile`.
- **Розумне сканування:** Додаток сам гортає список обраних відео.
- **Висока якість:** Завантаження через `yt-dlp`.

## Вимоги
- Python 3.8+
- Встановлені залежності

## Встановлення

1. Клонуйте репозиторій або завантажте файли.
2. Запустіть скрипт для налаштування (якщо ви вже це робили, пропустіть):
   ```powershell
   .\setup_venv.ps1
   ```
3. Активуйте віртуальне середовище:
   ```powershell
   .\venv\Scripts\activate
   ```

## Використання

Запустіть додаток:
```powershell
python -m src.main ваш_нікнейм --count 50
```

### Як це працює:
1. **Перший запуск:** Відкриється вікно браузера Chrome. Якщо ви не авторизовані — увійдіть у свій TikTok акаунт. 
2. **Сканування:** Після входу додаток перейде на сторінку ваших обраних відео і почне їх збирати. Не закривайте вікно, поки збір не завершиться.
3. **Завантаження:** Після збору посилань браузер закриється, і почнеться завантаження відео в папку `downloads/`.
4. **Наступні запуски:** Ваша сесія збережеться, тому входити знову (зазвичай) не доведеться.

## Налаштування
Ви можете змінити браузер для завантаження в `.env` (за замовчуванням `chrome` для `yt-dlp`).

## Структура проекту
- `src/scraper.py`: Новий логік збору посилань через Playwright.
- `src/downloader.py`: Логіка завантаження.
- `browser_profile/`: Папка з вашою сесією браузера (не видаляйте, щоб не входити знову).

## Відмова від відповідальності
Цей інструмент призначений лише для особистого використання. Будь ласка, дотримуйтесь умов використання TikTok.
